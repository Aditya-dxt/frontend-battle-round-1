**FrontEnd Battle 3.0** is the flagship event of the Web and Design Society (WebnD) at IIT Bhubaneswar — an intense online vibe-coding competition designed to test frontend development speed and excellence under pressure.

- 🎯 **Round 1 Task:** Recreate and enhance a given Vexel AI landing page design

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/Aditya-dxt/frontend-battle-round-1.git

# Navigate into the project
cd frontend-battle-round-1

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
